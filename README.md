## Projects

### 1. Drawing Tool
- Draw straight lines and rectangles by dragging the mouse.
- Practice working with **mouse events** and **interactive drawing**.

### 2. Cropping Tool
- Crop images exactly as you want with a simple interactive interface.
- Helps understand **image slicing** and **user interactions**.

### 3. Webcam Recorder
- Record videos directly from your webcam.
- Learn about **video capture**, **writing video files**, and **frame processing**.
